# CTF Writeups

This repository contains my personal Capture The Flag (CTF) writeups.  
Each writeup documents the full attack path including enumeration, exploitation, and privilege escalation where applicable.

---

## Platforms

- VulnHub  
- Hack The Box  
- TryHackMe  
- Other standalone CTF challenges  

---

## Structure

Each folder represents a machine or challenge and includes:

- Enumeration steps  
- Tools used  
- Exploitation process  
- Post-exploitation / Privilege escalation  
- Lessons learned  

---

## Disclaimer

All challenges were performed in legal lab environments for educational purposes only.
